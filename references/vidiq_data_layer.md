# vidIQ Data Layer

vidIQ is a data validation layer, not a content strategy by itself.

## Use vidIQ for

- keyword search volume
- competition score
- related keywords
- video SEO metadata checks
- competitor tracking
- content ideas and trends
- thumbnail/title inspiration

## Do not use vidIQ for

- final creative judgment
- actual retention performance
- legal or YPP decisions
- guaranteeing views
- replacing YouTube Studio

## Data hierarchy

```text
YouTube Studio first-party performance > actual comments > vidIQ external opportunity data > creator intuition
```

## Topic validation matrix

| Data pattern | Interpretation | Action |
|---|---|---|
| High search, low competition | strong search opportunity | prioritize if channel fit exists |
| High search, high competition | large market, hard entry | require strong differentiation |
| Low search, low competition | possible long-tail niche | check comments and adjacent topics |
| Low search, high competition | weak opportunity | usually skip |
| Search low, comments strong | community-driven topic | consider Shorts/Q&A/series |
| Search high, channel fit low | temptation topic | usually avoid |
