# First 30s Audit Rubric

Created by **Cruz Olli**

## The opening conversion path

A YouTube opening should convert a viewer through these checks:

1. Did I click the right video?
2. Is this for me?
3. Is the creator going to answer the title promise?
4. Did I already get value?
5. Is there a reason to keep watching?

## Timestamp targets

| Timestamp | Required job |
|---|---|
| 0-3s | Direct pain, contradiction, result, or stakes |
| 3-8s | Confirm the title/thumbnail promise |
| 8-15s | Deliver first concrete value |
| 15-25s | Open curiosity loop and show structure |
| 25-30s | Transition into first section |

## Common drop causes

- Greeting before value.
- Logo animation before relevance.
- Explaining why the topic matters before saying what the viewer gets.
- Title says one thing; opening starts somewhere else.
- First value appears after 20 seconds.
- Static visual track with low information density.
- Too much channel meta: “I got many comments asking...”
- No specific target viewer.

## Fix patterns

- Move the strongest sentence from 0:20-0:30 to 0:00-0:05.
- Replace greeting with a diagnostic claim.
- Add a number, contrast, or exact mistake before 0:08.
- Make the first sentence reuse the title’s main keyword.
- Add a text overlay before 0:05.
- Add visual proof before 0:10 when making a performance/data claim.
